Use tags to checkout parts of the above tutorial.

## Prerequisites

Below is for *macOS*, but should be something similar for Linux or Windows.

```shell script
$ brew install cmake ninja
```

## Build / Run

```shell script
$ ./build.sh
$ ./chuck-says
```
